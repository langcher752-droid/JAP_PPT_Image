# 日语词汇PPT图片增强工具 - Web前端

这是Web版本的前端页面，可以部署到GitHub Pages。

## 使用方法

1. 将 `index.html` 上传到GitHub仓库
2. 在GitHub仓库设置中启用GitHub Pages
3. 修改 `index.html` 中的API服务器地址为你的服务器地址
4. 访问你的GitHub Pages地址即可使用

## 功能特点

- 📤 拖拽上传PPT文件
- 📊 实时显示处理进度
- 📝 显示处理日志
- 💾 下载处理后的PPT
- 🔧 可配置API服务器地址

## 技术栈

- 纯HTML + CSS + JavaScript
- 无需构建工具，直接部署即可

## 注意事项

- 确保API服务器已正确部署并运行
- 建议使用HTTPS访问API服务器
- 文件大小限制：100MB
