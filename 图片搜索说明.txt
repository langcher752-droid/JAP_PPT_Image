图片搜索功能说明
================

当前使用的图片源：
-----------------
程序使用 Picsum Photos (Lorem Picsum) 作为图片源。
这是一个稳定的图片服务，提供高质量的随机图片。

注意：
-----
1. 当前版本使用的图片是随机的，可能与日语词汇不完全相关
2. 这是为了确保程序能够稳定运行（避免网络连接问题）
3. 图片质量很好，适合作为背景使用

如果需要更相关的图片：
---------------------
方案1：使用Pixabay API（推荐）
---------------------------
1. 访问 https://pixabay.com/api/docs/
2. 注册账号并获取免费的API key
3. 修改main.py中的search_images方法，添加您的API key
4. 这样就能搜索到与关键词相关的图片

方案2：使用Pexels API
-------------------
1. 访问 https://www.pexels.com/api/
2. 注册账号并获取API key
3. 修改代码使用Pexels API

方案3：手动添加图片
-----------------
1. 准备与每个词汇相关的图片
2. 将图片放在一个文件夹中
3. 修改代码，让它从本地文件夹读取图片

方案4：使用Bing图片搜索（需要处理反爬虫）
------------------------------------
可以修改代码使用Bing图片搜索API，但需要：
1. 注册Azure账号
2. 获取Bing Search API key
3. 修改代码使用该API

当前版本的优点：
--------------
✓ 稳定可靠，不会因为API限制而失败
✓ 图片质量高，适合作为PPT背景
✓ 无需注册任何服务
✓ 下载速度快
