GUI版本使用说明
================

功能特点：
---------
✓ 图形化界面，操作简单直观
✓ 支持拖拽PPT文件（需要安装tkinterdnd2）
✓ 实时显示处理日志
✓ 可选择导出文件夹
✓ 支持停止处理

安装步骤：
---------

1. 安装GUI所需依赖：
   双击运行"安装GUI依赖.bat"
   或手动运行：
   pip install -r requirements-gui.txt -i https://pypi.tuna.tsinghua.edu.cn/simple --user

2. 运行GUI程序：
   双击运行"运行GUI.bat"
   或手动运行：
   python gui.py

使用方法：
---------

1. 选择PPT文件：
   - 方式1：将PPT文件拖拽到拖拽区域
   - 方式2：点击"选择PPT文件"按钮，浏览选择文件

2. 选择导出文件夹（可选）：
   - 点击"选择文件夹"按钮
   - 如果不选择，文件将保存在PPT原文件所在目录

3. 开始处理：
   - 点击"开始处理"按钮
   - 处理过程中可以在日志区域查看进度
   - 可以随时点击"停止"按钮中断处理

4. 查看结果：
   - 处理完成后会弹出提示框
   - 处理后的PPT文件名格式：原文件名_enhanced.pptx

界面说明：
---------

【PPT文件区域】
- 拖拽区域：将PPT文件拖到这里
- 文件路径显示：显示当前选择的文件
- 选择PPT文件按钮：点击选择文件

【导出设置区域】
- 导出文件夹：显示选择的导出文件夹
- 选择文件夹按钮：点击选择导出位置

【处理日志区域】
- 实时显示处理进度和日志信息
- 不同颜色的日志：
  * 绿色：成功信息
  * 红色：错误信息
  * 黄色：警告信息
  * 蓝色：调试信息
  * 青色：一般信息

【控制按钮】
- 开始处理：开始处理PPT
- 停止：中断当前处理
- 清空日志：清空日志显示区域

注意事项：
---------
1. 拖拽功能需要安装tkinterdnd2库
2. 处理过程中请勿关闭程序窗口
3. 如果遇到错误，请查看日志区域的错误信息
4. 确保网络连接正常（用于下载图片）

故障排除：
---------
Q: 拖拽功能不可用？
A: 运行：pip install tkinterdnd2

Q: 程序无法启动？
A: 检查是否已安装所有依赖库

Q: 处理失败？
A: 查看日志区域的错误信息，检查：
   - PPT文件是否损坏
   - 网络连接是否正常
   - 是否有足够的磁盘空间

Q: 图片下载失败？
A: 检查网络连接，或配置Google API（查看GoogleAPI配置说明.txt）
